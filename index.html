Tentang Saya

Ayo selalu semangat pelajari hal-hal baru dan jangan pernah menyerah!

Saya terobsesi untuk membuat sesuatu dan bahkan lebih terobsesi untuk membuat sesuatu menjadi lebih baik. Saya berkecimpung dalam dunia web programing sejak saya umur 16 tahun, saya terus mencoba dan mencari tahu bagaimana cara membuat website dengan baik dan benar. Dengan project pertama saya yaitu membuat website statis form sederhana dengan menggunakan bahasa HTML & CSS. Dengan berjalannya waktu saya pun terus mencoba bergerak dibidang web programing karena menurut saya membuat website itu seru dan menyenangkan. Saya adalah desain dan pengembangan web front-end, membuat piksel keajaiban di Figma dan mengubahnya menjadi HTML & CSS semantik yang indah.
